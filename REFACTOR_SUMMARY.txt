TUY CHINH
================================================================================

Thay doi so anh hien thi:

  Trong CSS (styles.css):
  .gallery-item {
      flex: 0 0 calc(100% / 3);  /* 3 anh - thay doi so nay */
  }

  Vi du:
  - calc(100% / 2) = 2 anh
  - calc(100% / 4) = 4 anh
  - calc(100% / 5) = 5 anh

Thay doi toc do scroll:

  Trong JS (script.js):
  const scrollAmount = itemWidth * 3; // Scroll 3 items - thay doi so nay

  Vi du:
  - itemWidth * 1 = Scroll 1 anh
  - itemWidth * 5 = Scroll 5 anh

Thay doi mau nut:

  Trong CSS:
  .carousel-btn {
      color: var(--primary-color);  /* Thay doi mau */
  }
